
# Hourly Rain

Minute weather measurements

## Structure

`HourlyRain`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `m_1_h` | `float` | Optional | (where available) Precipitation, mm/h. Please note that only mm/h as units of measurement are available for this parameter |
