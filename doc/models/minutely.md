
# Minutely

Minute weather measurements

## Structure

`Minutely`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `dt` | `float` | Optional | Time of the forecasted data, unix, UTC |
| `precipitation` | `float` | Optional | Precipitation, mm/h. Please note that only mm/h as units of measurement are available for this parameter. |

## Example (as JSON)

```json
{
  "dt": 1684929540.0,
  "precipitation": 0.0
}
```
